# 新格 · 新标签页 · NovaTab（Chrome 扩展）

玻璃质感的自定义新标签页，由玻璃拟态设计原型（见 `../docs/原型设计参考-离线版.html`）落地为标准 Manifest V3 Chrome 扩展。

## 功能

- **时钟 / 日期 / 问候语**：实时时钟（时:分:秒）、中文日期与星期、按时段问候。
- **实时天气**：天气数据由 [Open-Meteo](https://open-meteo.com)（免 API Key）提供。**自动定位**优先使用浏览器定位服务（精确，不用 IP，避免代理导致错位），由经纬度反查城市名（BigDataCloud 免 Key）；未授权时弹通知询问是否开启定位。也可关闭自动定位手动指定城市。点击天气芯片手动刷新，结果缓存 30 分钟。
- **多引擎搜索**：Google / 百度 / Bing / DuckDuckGo / 站内书签 / 自定义引擎；引擎下拉切换；回车搜索；`/` 快捷聚焦。
- **本地联想**：输入时即时联想本地快捷方式，方向键选择、回车直达。
- **快捷方式网格**：默认站点、字母色块图标或自定义图标；添加站点时**自动抓取网站 favicon 并内嵌**。
- **文件夹**：拖拽合并生成文件夹、**拖图标到文件夹任意位置即放入**、**空白处右键新建文件夹**、弹层管理、移出 / 删除 / 重命名 / 解散。
- **拖拽**：顶层排序、合并成夹、拖到末尾。
- **主题**：浅色 / 深色 / 跟随系统，顶栏一键切换。
- **壁纸**：6 套内置渐变 + 自定义上传 + **在线高清壁纸**（[Wallhaven](https://wallhaven.cc) 免 Key）：可选风格（自然 / 城市 / 太空 / 极简 / 抽象 / 动漫等 9 种）、自动或手动更换、自定义更换频率（每次打开 / 30 分钟 / 1 小时 / 6 小时 / 每天）、「立即换一张」。
- **外观微调**：毛玻璃、图标圆角、每行图标数。
- **数据备份**：导出 / 导入 JSON（含图标数据）、恢复默认。
- **本地数据文件（防丢失 / 跨浏览器）**：在设置中绑定一个本地 `.json` 文件，所有更改自动写入；换浏览器或重装插件后用「从文件恢复」找回（基于 File System Access API）。**首次安装**会在右下角弹出引导，说明利弊后由你选择是否绑定。
- 全部配置持久化于 `localStorage`。

## 安装（加载未打包扩展）

1. 打开 Chrome，地址栏输入 `chrome://extensions`。
2. 右上角开启「开发者模式」。
3. 点击「加载已解压的扩展程序」，选择本 `extension/` 目录。
4. 新开标签页即为新格 NovaTab。

> 天气、定位、图标抓取、在线壁纸需要联网。`manifest.json` 的 `host_permissions` 已声明：Open-Meteo（天气）、BigDataCloud（经纬度反查城市）、Google s2（网站图标）、Wallhaven（在线壁纸）。浏览器定位会在首次使用时弹出 Chrome 原生授权提示，请点「允许」。

## 相对设计稿的优化

| 项 | 设计稿 | 本扩展 |
|---|---|---|
| 布局 | 残留未接入的侧边栏 CSS | 精简为居中布局，仅保留左下角设置入口 |
| 天气 | 写死「23° 多云 北京」 | 真实 API + 浏览器定位 + 城市可改 + 开关 + 刷新 |
| 壁纸 | 6 套渐变 + 上传 | 增加在线高清壁纸（风格/自动更换/频率） |
| 添加站点图标 | 仅字母/手动上传 | 按网址自动抓取并内嵌 favicon |
| 自定义引擎/重命名/重置 | 原生 `prompt`/`confirm` | 统一风格的应用内弹窗 |
| 搜索 | 无联想 | 本地快捷方式联想 |
| 数据 | 仅 localStorage | 导出/导入 + 可绑定本地文件自动同步 |
| 形态 | 离线单 HTML | 标准 MV3 扩展（newtab 覆盖） |
